# HEARTBEAT.md

## Hourly Briefs (Heartbeat)

**Goal:** Every hour (approximately), send Caligula a brief.

### Quiet hours
- Do **not** send routine briefs between **12:00 AM – 4:00 AM ET** unless urgent.

### Brief format (Both)
1) **Scoreboard**
   - Website: status (live?), last deploy, CTA status
   - Outreach: leads added/contacted, replies, meetings booked
   - Build: tasks shipped, blockers
2) **3 bullets**
   - Wins
   - Risks
   - Next moves (next hour)

### Execution notes
- If nothing changed since last brief, stay quiet.
- If something is urgent, send immediately (ignore quiet hours).
